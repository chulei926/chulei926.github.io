# 去除所有的html标签
```javascript
<\/?.+?>
```

# 去除html所有标签中的style="..."属性
```javascript
\s*style="[^"]*"
```

# 去除html标签中除指定标签之外的其他所有标签
```javascript
<(?!(sup|sub|\/sup|\/sub)).*?>  // 去除除了sub和sup之外的所有html标签
```

# 去除html标签中最后一个指定的标签
```javascript
(<br ?/?>)$                     // 去除最后一个<br>标签。 
```
