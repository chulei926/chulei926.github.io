# 查看内存使用，以G为单位
```shell
$ free -g
```

# 查看内存使用，以M为单位

```shell
$ free -m
```

# 内存占用高的前20

```shell
$ ps aux | head -1;ps aux |grep -v PID |sort -rn -k +4 | head -20
```

# 查看进程的内存占用

```shell
$ pidstat -r -p 24427 1 5 
```
