# 2. 类加载子系统

![类加载子系统](/images/jvm/类加载子系统.png)

类加载子系统分为三步：
1. 加载阶段
    - 引导类加载器 bootstrap classloader
    - 扩展类加载器 extension classloader
    - 应用类加载器 application classloader
2. 链接阶段
    - 验证 verify
    - 准备 prepare
    - 解析 resolve
3. 初始化阶段
    - 静态变量的显示初始化
